# BlueDOG

BlueDOG is a simple programming language written in C.

## Installation

To install BlueDOG:

1. Clone the repository:
   ```bash
   git clone https://github.com/yucchannel/bluedog.git
   
2. Go to BlueDOG folder:
   ```bash
   cd BlueDOG
